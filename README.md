- 修正 header：在手機上分兩行顯示（上：標題；下：列印按鈕＋可橫向捲動的分頁籤），不再重疊。

## 部署
1. 將 `index.html` 上傳至 GitHub repo 根目錄（覆蓋舊檔）。
2. 到 **Settings → Pages** 啟用 Pages（Deploy from a branch → `main` / root）。
3. 打開 Pages 網址即可使用。
